## Overview

**QuantSignal** is a live-analysis trading assistant that combines:

- real-time market data
- live candlestick charts
- rule-based signal analysis
- AI-generated reasoning
- risk and invalidation logic
- trade setup explanation in plain English

The goal is **not** to promise perfect price prediction.  
The goal is to help users **validate live setups**, understand market context, and make more structured decisions.

---

## Key Features

- **Live crypto charting**
  - Real-time candle updates
  - Symbol switching
  - Timeframe switching
- **Live signal analysis**
  - Buy / Sell / Hold / Avoid
  - Confidence score
  - Risk score
  - Regime detection
- **Trade structure generation**
  - Stop loss
  - Target
  - Invalidation zone
- **AI explanation layer**
  - Human-readable setup summary
  - Trade logic explanation
  - Risk-aware commentary
- **Multi-input workflow**
  - Live chart mode
  - Manual setup analysis
  - Screenshot upload
  - Trade CSV upload
- **Dashboard and history**
  - Signal history
  - Analysis runs
  - Basic performance review

---

## Why QuantSignal?

Most tools are either:

- only charting platforms
- only backtesting platforms
- only AI chat layers
- or too complex for practical live use

**QuantSignal** is designed to sit between charting and decision support:

> live market data + structured signal logic + AI explanation + risk controls

---

## Product Philosophy

QuantSignal is built around these principles:

- **live market awareness**
- **clear signal reasoning**
- **risk-first analysis**
- **modular agent-like workflow**
- **AI as assistant, not blind oracle**

This project is inspired by **multi-agent reasoning patterns** and uses a modular pipeline for:

- ingestion
- signal generation
- regime detection
- risk scoring
- decision output
- plain-English explanation
